# LiaoKit 组件文档索引

本目录包含 LiaoKit 组件库的所有组件使用指南。每个组件文档提供了详细的使用说明、属性列表、事件说明、插槽信息以及高级用法示例。

## 核心组件

- [LiaoApp](./LiaoApp.md) - 核心容器组件，提供完整聊天应用布局和功能整合
- [LiaoWindow](./LiaoWindow.md) - 聊天窗口容器组件，提供标准窗口布局和控制
- [LiaoWindowHeader](./LiaoWindowHeader.md) - 窗口头部组件，支持标题和操作按钮

## 消息相关组件

- [LiaoMessageList](./LiaoMessageList.md) - 消息列表组件，展示聊天历史记录
- [LiaoMessageBubble](./LiaoMessageBubble.md) - 消息气泡组件，用于文本消息展示
- [LiaoTextBubble](./LiaoTextBubble.md) - 文本气泡组件，专用于文本消息的渲染和交互
- [LiaoImageBubble](./LiaoImageBubble.md) - 图片消息气泡组件，用于图片消息展示
- [LiaoPluginBubble](./LiaoPluginBubble.md) - 插件化气泡组件，支持自定义内容卡片

## 输入相关组件

- [LiaoInputArea](./LiaoInputArea.md) - 消息输入区域组件，支持文本输入和各种操作

## 智能功能组件 ✨ 新增

- [AiMessageAdapter](./AiMessageAdapter.md) - AI智能消息格式适配器，自动转换各种消息格式为标准格式

## 工具组件

- [LiaoIcon](./LiaoIcon.md) - 图标组件，提供统一的图标使用方式
- [LiaoQuickActionBar](./LiaoQuickActionBar.md) - 快捷操作栏组件，提供常用操作按钮

## 插件组件

- [LiaoFilePreview](./LiaoFilePreview.md) - 文件预览插件，支持多种文件格式预览
- [LiaoProgressPlugin](./LiaoProgressPlugin.md) - 进度插件，用于展示任务进度
- [LiaoMediaCarouselPlugin](./LiaoMediaCarouselPlugin.md) - 媒体轮播插件，支持图片和视频轮播
- [LiaoTimelinePlugin](./LiaoTimelinePlugin.md) - 时间线插件，展示时间序列事件
- [LiaoVotePlugin](./LiaoVotePlugin.md) - 投票插件，支持多种投票形式
- [LiaoActionsPlugin](./LiaoActionsPlugin.md) - 操作插件，提供各种交互操作
- [LiaoStatsPlugin](./LiaoStatsPlugin.md) - 统计插件，展示数据统计信息
- [LiaoInfoPlugin](./LiaoInfoPlugin.md) - 信息插件，展示格式化信息内容
