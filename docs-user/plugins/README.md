# 插件总览 (Plugins)

LiaoKit 的消息流插件系统。这些插件可以作为消息类型嵌入到聊天窗口中。

| 名称 | 描述 | 更新时间 |
| :--- | :--- | :--- |
| [LiaoActionsPlugin（操作栏插件）](./liao-actions-plugin.md) | 提供常用操作入口，配合输入与消息交互 | 2025-12-18 |
| [LiaoFaqCardPlugin（FAQ 卡片插件）](./liao-faq-card-plugin.md) | 以卡片形式展示常见问题与答案 | 2025-12-18 |
| [LiaoFormPlugin（表单插件）](./liao-form-plugin.md) | 在消息中收集结构化数据并提交 | 2025-12-18 |
| [LiaoImageCardPlugin（图片卡片插件）](./liao-image-card-plugin.md) | 以卡片形式展示图片与说明 | 2025-12-18 |
| [LiaoInfoPlugin（信息插件）](./liao-info-plugin.md) | 结构化信息展示与动作触发 | 2025-12-18 |
| [LiaoListPlugin（列表插件）](./liao-list-plugin.md) | 列表项选择与“更多”操作，常用于导航与选项 | 2025-12-18 |
| [LiaoMediaCarouselPlugin（媒体轮播插件）](./liao-media-carousel-plugin.md) | 图片/视频等媒体轮播展示 | 2025-12-18 |
| [LiaoPluginDebugger（插件调试器）](./liao-plugin-debugger.md) | 可视化调试插件状态与事件流 | 2025-12-18 |
| [LiaoProgressPlugin（进度插件）](./liao-progress-plugin.md) | 展示任务或流程进度，支持与消息/操作联动 | 2025-12-18 |
| [LiaoStatsPlugin（统计插件）](./liao-stats-plugin.md) | 展示统计数据与趋势，可联动图表 | 2025-12-18 |
| [LiaoTimelinePlugin（时间线插件）](./liao-timeline-plugin.md) | 以时间线形式展示事件流与状态 | 2025-12-18 |
| [LiaoVotePlugin（投票插件）](./liao-vote-plugin.md) | 提供投票与结果展示，支持单选/多选 | 2025-12-18 |
