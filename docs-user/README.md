# LiaoKit 使用文档（User Guide）

> 本模块面向“使用者”，用于指导如何集成与使用 LiaoKit 组件库。与 `@Docs` / `src/docs` 的“开发向”文档（实现细节、内部规约）严格区分。

## 你将获得
- 完整的阅读路径：概览 → 快速开始 → 组件/插件详解 → 场景组合 → FAQ/重要事项 → 扩展展望
- 统一的排版规范：Props/Events/Slots/Theme/A11y/性能/排查
- 可演进的目录：后续支持 i18n 与静态站点

## 如何阅读
- 第一次使用：先读“快速开始”，跑通最小示例，再按需深入组件或插件文档
- 面向场景：直接进入“场景组合秘籍”，选与业务接近的案例按步骤集成
- 碰到问题：优先查看“FAQ”，并对照“Troubleshooting”自检

## 与内部文档的关系
- `@Docs`：团队开发用示例与指南，包含实现细节与调试说明
- `src/docs`：开发向文档（如自定义插件、示例集合）
- `docs-user`：使用者视角，聚焦 API 使用、组合模式与最佳实践
