# 场景组合索引

## 如何选择
- 明确业务目标 → 选择合适的窗口/消息/插件组合 → 逐步集成
- 优先采用“最小可行组合”，按需迭代增强

## 示例场景
- 客服咨询流程（Actions + MessageList + Progress）\n- 订单处理工作流（Timeline + Stats + FilePreview）\n- FAQ 问答机器人（AI Adapter + MessageBubble）\n\n> 进入对应场景文档查看步骤与代码片段。\n
