# 重要事项与常见问题（FAQ）

## 安装与构建
- 版本不兼容：核对组件库与框架版本，参考 `CHANGELOG.md`\n- 构建失败：检查打包工具配置、类型声明与样式入口\n

## 样式与冲突
- 样式未生效：确保主样式与变量文件已引入\n- 冲突与污染：避免全局 Reset 影响组件内部样式作用域\n

## 事件与状态
- 事件未触发：校验回调签名与组件/插件生命周期\n- 状态异常：检查会话/窗口管理逻辑与数据来源\n

## 性能优化
- 长列表：使用虚拟滚动或分页，降低一次性渲染\n- 媒体资源：按需加载与压缩，避免首屏压力\n

## 无障碍与国际化
- 键盘操作与语义化标签：遵循 A11y 建议\n- 国际化：后续可引入 i18n 目录并保持文档结构一致\n

## 诊断清单
- 逐步排查：安装 → 样式 → 引入 → 事件 → 状态 → 渲染 → 资源\n- 复现与最小化：构建最小示例，定位问题源头\n
