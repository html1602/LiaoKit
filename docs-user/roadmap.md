# 扩展方向展望（Roadmap）

## 近期（小版本）
- 文档完善与示例补齐（组件/插件/场景）\n- Props/Events 表优化与半自动生成\n

## 中期（能力扩展）
- 更完整的主题系统与设计令牌\n- 更丰富的插件生态与组合模板\n

## 长期（生态与兼容）
- 多语言文档与站点化构建\n- 与更多构建工具/框架的兼容\n

## 接口稳定性与变更策略
- 以向后兼容为原则，破坏性变更需提前公告并提供迁移指南\n- 严格在 `CHANGELOG.md` 标注变更与影响范围\n
